# KPI_data_quality_laboratory
Developed a Python-based automation system to extract, process, and analyze quality control data from AU480 clinical analyzer using Selenium and Pandas. Implemented automated data pipelines (web scraping → cleaning → structured storage → analytical output) to improve laboratory performance monitoring and reduce manual data entry errors.
